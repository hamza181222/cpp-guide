Introduction to C++
The language that powers the world's most critical software

What is C++?

C++ is a powerful, general-purpose programming language created by Bjarne Stroustrup in 1979 as an extension of C. It combines low-level memory control with high-level abstractions, making it one of the most versatile languages ever created.

⚡ High Performance 🔧 System Programming 🎮 Game Development 🏆 Competitive Programming 🤖 Embedded Systems
Why Learn C++?
⚡ Speed & Performance
C++ compiles directly to machine code, making it one of the fastest languages. It's used where performance is critical.
🔧 Control Over Memory
Manual memory management gives you fine-grained control that languages like Python can't offer.
🏆 Competitive Programming
The #1 language in competitive programming due to its STL, speed, and extensive support in judges.
🎮 Game Development
Unreal Engine and most AAA game engines are built in C++. It's the language of the gaming industry.
🤖 AI & ML Backends
TensorFlow, PyTorch cores, and most AI frameworks are written in C++ for performance.
💡 Foundation for All
Learning C++ makes every other language easier. You'll deeply understand how programs work.

Switch Case

Use switch when comparing one variable against multiple constant values — cleaner than long if-else chains.

C++
int day = 3;

switch (day) {
    case 1: cout << "Monday";    break;
    case 2: cout << "Tuesday";   break;
    case 3: cout << "Wednesday"; break;
    case 4: cout << "Thursday";  break;
    case 5: cout << "Friday";    break;
    default: cout << "Weekend";
}
// Output: Wednesday
⚠️ Warning: Always use break to prevent fall-through to the next case.

Loops
Repeating actions efficiently
for Loop
C++
// Classic for loop
for (int i = 0; i < 5; i++) {
    cout << i << " ";      // 0 1 2 3 4
}

// Range-based for (C++11)
vector<int> v = {10, 20, 30};

while Loop
C++
int n = 1;
while (n <= 1024) {
    cout << n << " ";
    n *= 2;
}
// 1 2 4 8 16 32 64 128 256 512 1024
do…while Loop

Executes the body at least once, then checks the condition.

C++
int num;
do {
    cout << "Enter a positive number: ";
    cin  >> num;
} while (num <= 0);
Nested Loops
C++
// Multiplication table
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        cout << i * j << "\t";
    }
    cout << "\n";
}
// 1  2  3
// 2  4  6
// 3  6  9
break & continue
C++
// break — exits the loop entirely
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    cout << i;              // 0 1 2 3 4
}

// continue — skips current iteration
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) continue;
    cout << i;              // 1 3 5 7 9
}

Functions
Reusable blocks of logic
Function Declaration & Definition
C++
// Declaration (prototype)
int add(int a, int b);

// Definition
int add(int a, int b) {
    return a + b;
}

int main() {
    cout << add(3, 5);    // 8
    return 0;
}
Parameters & Return Values
C++
// Pass by value  — copy is made
void doubleVal(int x) { x *= 2; }     // x outside unchanged

// Pass by reference — modifies original
void doubleRef(int &x) { x *= 2; }     // x outside IS changed

// Default parameters
int power(int base, int exp = 2) {
    int result = 1;
    for (int i = 0; i < exp; i++) result *= base;
    return result;
}
cout << power(3);       // 9  (exp defaults to 2)
cout << power(2, 8);    // 256
Function Overloading

Same function name, different parameter types or counts.

C++
int    area(int side)                  { return side * side; }
double area(double r)                   { return 3.14159 * r * r; }
int    area(int len, int wid)           { return len * wid; }

area(5);          // 25     (int)
area(3.0);        // 28.27  (double)
area(4, 6);       // 24     (rectangle)
Recursion
C++
// Factorial: n! = n × (n-1)!
int factorial(int n) {
    if (n <= 1) return 1;    // base case
    return n * factorial(n - 1);
}

// Fibonacci
int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}

cout << factorial(5);  // 120
cout << fib(10);       // 55
⚠️ Warning: Every recursive function needs a base case to prevent infinite recursion and stack overflow.

Searching & Sorting
C++
#include <algorithm>
using namespace std;

int arr[] = {5, 2, 8, 1, 9, 3};
int n = 6;

// Sort (uses introsort, O(n log n))
sort(arr, arr + n);                    // {1,2,3,5,8,9}
sort(arr, arr + n, greater<int>());    // {9,8,5,3,2,1}

// Binary search (array must be sorted)
sort(arr, arr + n);
bool found = binary_search(arr, arr+n, 8);  // true

// Linear search
for (int i = 0; i < n; i++)
    if (arr[i] == 8) { cout << "Found at " << i; break; }

Common String Problems
C++
// Check palindrome
bool isPalindrome(string s) {
    int l = 0, r = s.size() - 1;
    while (l < r)
        if (s[l++] != s[r--]) return false;
    return true;
}

// Count vowels
int countVowels(string s) {
    int cnt = 0;
    string vowels = "aeiouAEIOU";
    for (auto c : s)
        if (vowels.find(c) != string::npos) cnt++;
    return cnt;
}

Pointers
Variables that store memory addresses
Pointer Basics
C++
int  x   = 10;
int* ptr = &x;       // ptr holds address of x

cout << x;           // 10    (value)
cout << &x;          // 0x... (address of x)
cout << ptr;         // 0x... (same address)
cout << *ptr;        // 10    (dereference → value at address)

*ptr = 99;
cout << x;           // 99    (x was changed through pointer!)
Dynamic Memory
C++
// Allocate single int on heap
int* p = new int(42);
cout << *p;           // 42
delete p;             // ALWAYS free heap memory

// Dynamic array
int n = 5;
int* arr = new int[n];
for (int i = 0; i < n; i++) arr[i] = i + 1;
delete[] arr;         // use delete[] for arrays
⚠️ Every new must have a matching delete. Forgetting this causes memory leaks.
Pointers with Arrays
C++
int arr[] = {10, 20, 30};
int* p = arr;          // p points to arr[0]

cout << *p;            // 10
cout << *(p+1);        // 20 (pointer arithmetic)
cout << *(p+2);        // 30

// Traverse with pointer
for (int i = 0; i < 3; i++)
    cout << *(p + i) << " ";

Modeling the real world with classes and objects
Classes & Objects
C++
class Car {
public:
    string brand;
    int    speed;

    void accelerate(int amount) {
        speed += amount;
        cout << brand << " → " << speed << " km/h\n";
    }
};

int main() {
    Car c;
    c.brand = "BMW";
    c.speed = 0;
    c.accelerate(100);   // BMW → 100 km/h
}
Constructors & Destructors
C++
class Student {
private:
    string name;
    int    age;

public:
    // Constructor
    Student(string n, int a) : name(n), age(a) {
        cout << name << " enrolled!\n";
    }

    // Destructor
    ~Student() {
        cout << name << " graduated!\n";
    }

    // Getter (encapsulation)
    string getName() const { return name; }
};

Student s("Alice", 20);   // Alice enrolled!
// When s goes out of scope → Alice graduated!
Inheritance

// Polymorphism via base pointer
Animal* a = new Dog();

Standard Template Library
Ready-made data structures and algorithms
vector — Dynamic Array
C++
#include <vector>
vector<int> v = {1, 2, 3};
v.push_back(4);      // {1,2,3,4}
v.pop_back();        // {1,2,3}
v.size();            // 3
v.front(); v.back(); // first and last element
sort(v.begin(), v.end());
map, set, pair
C++
#include <map>
#include <set>

// map — key-value, sorted by key
map<string, int> freq;
freq["apple"]++;
freq["banana"] = 3;
for (auto [k, v] : freq) cout << k << ": " << v << "\n";

// set — unique sorted elements
set<int> s = {3, 1, 4, 1, 5};  // {1,3,4,5}
s.insert(2);               // {1,2,3,4,5}
s.erase(3);                // {1,2,4,5}
s.count(4);                // 1 (exists)

// pair
pair<int, string> p = {1, "hello"};
cout << p.first << " " << p.second;
stack, queue, priority_queue, deque
C++
#include <stack>
#include <queue>
#include <deque>

// Stack (LIFO)
stack<int> st;
st.push(1); st.push(2);
st.top();   // 2
st.pop();   // removes 2

// Queue (FIFO)
queue<int> q;
q.push(10); q.push(20);
q.front();  // 10
q.pop();    // removes 10

// Priority Queue (max-heap by default)
priority_queue<int> pq;
pq.push(5); pq.push(1); pq.push(9);
pq.top();   // 9 (max element)

// Min-heap
priority_queue<int, vector<int>, greater<int>> minpq;
Common STL Algorithms
C++
#include <algorithm>
#include <numeric>
vector<int> v = {3,1,4,1,5,9,2};

sort(v.begin(), v.end());              // sort ascending
reverse(v.begin(), v.end());          // reverse
unique(v.begin(), v.end());           // remove consecutive dups
accumulate(v.begin(), v.end(), 0);    // sum
*max_element(v.begin(), v.end());      // max value
*min_element(v.begin(), v.end());      // min value
count(v.begin(), v.end(), 1);         // count occurrences
find(v.begin(), v.end(), 5);          // iterator to element
next_permutation(v.begin(), v.end());  // generate permutations

Time Complexity Guide
nMax ComplexityAlgorithm Examples
≤ 10O(n!)All permutations, brute force
≤ 20O(2ⁿ)Bitmask DP, subset enumeration
≤ 500O(n³)Floyd-Warshall, all-pairs shortest path
≤ 5000O(n²)Bubble/insertion sort, O(n²) DP
≤ 10⁶O(n log n)Merge sort, binary search, segment tree
≤ 10⁸O(n)Linear scan, prefix sums
≤ 10¹⁸O(log n)Binary search on answer, matrix exponentiation

Debugging Tips
🐛 Runtime Error (RE)
Check array bounds, null pointers, division by zero, and stack overflow from deep recursion.
⏱️ Time Limit (TLE)
Check your complexity. Enable fast I/O. Remove unnecessary endl (use "\n").
❌ Wrong Answer (WA)
Test edge cases: n=0, n=1, all same elements, negative numbers, max constraints.
💾 Memory Limit (MLE)
Avoid storing O(n²) data when O(n) suffices. Use rolling arrays in DP.
