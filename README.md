# C++ Guide Website 📘

A beginner-friendly educational website for learning C++ programming step by step using simple explanations and practical code examples.

👉 Live Demo: https://hamza181222.github.io/cpp-guide/

---

## 🚀 About the Project

This website is designed to help beginners learn C++ in an easy and structured way.

Each topic includes:
- A clear explanation of the concept
- A simple C++ code example
- Output explanation to understand how the program works

The goal is to connect theory with practice to make learning programming easier.

---

## 🧠 What You Will Learn

- Variables & Data Types  
- Input & Output (cin / cout)  
- Conditional Statements (if / else)  
- Loops (for / while)  
- Functions  
- Arrays  

---

## 📌 Example Structure (Inside Each Topic)

Each lesson follows this format:

### 🧠 Explanation
A simple description of the concept.

### 💻 Code Example
A working C++ program.

### 📌 Output
Shows the result of the program based on sample input.
